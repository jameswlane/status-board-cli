# Usage
<!-- usage -->
```sh-session
$ npx status-board-cli COMMAND
running command...
```
To install
```sh-session
$ npm install -g status-board-cli
$ status-board COMMAND
running command...
$ status-board (-v|--version|version)
status-board-cli/0.0.0 darwin-x64 node-v8.11.1
$ status-board --help [COMMAND]
USAGE
  $ status-board COMMAND
...
```
